# Youtube / Multimedia Converter (Bash + Zenity + yt-dlp)

Skrypt **converter.sh** umożliwia pobieranie materiałów audio oraz wideo z serwisów takich jak YouTube, korzystając z narzędzia **yt-dlp**.  
Interfejs graficzny został zrealizowany przy użyciu **Zenity**, dzięki czemu obsługa jest prosta i intuicyjna.


## Funkcje

* Pobieranie **audio (MP3)** lub **wideo (MP4)**
* Graficzne okna dialogowe (Zenity) 
* Obsługa logów w katalogu tymczasowym 
* Plik konfiguracyjny `.rc` 
* Manual w formacie `man`
